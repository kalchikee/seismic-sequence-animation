About this project
The Ridgecrest sequence activated ~50 km of unmapped fault strands in the Eastern California Shear Zone. This pipeline queries the USGS FDSN API, fits Omori decay parameters, maps b-value spatial variations, and produces depth cross-sections revealing the seismogenic zone geometry.
